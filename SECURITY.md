# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly via [GitHub private vulnerability reporting](https://github.com/deepjoy/shoebox-webapp/security/advisories/new).

**Please do not open a public issue for security vulnerabilities.**

## What to expect

- Acknowledgement within 48 hours.
- A fix or mitigation plan within 7 days for confirmed issues.
- Credit in the release notes (unless you prefer to remain anonymous).
